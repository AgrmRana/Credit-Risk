# Model Validation Framework

This platform validates probability of default models through out-of-sample holdout testing and five-fold stratified cross validation.

Reported measures include ROC AUC, PR AUC, KS statistic, Gini coefficient, precision, recall, F1, confusion matrix, calibration curve, lift chart, gain chart, and threshold optimization. Metrics are generated by `credit_risk_platform.training.train` and stored in `artifacts/metrics.json`.

The champion model is selected by holdout ROC AUC after each candidate has been tuned by randomized cross-validation. This is intentionally transparent and conservative for an internal model-risk review.
